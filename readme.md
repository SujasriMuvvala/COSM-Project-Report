# COSM Project – Data Analysis using Python

This project involves exploratory data analysis on a real-world dataset using Python.
The objective is to analyze, clean, and visualize data to extract meaningful insights.

## Dataset
- File: `penguins.csv`
- The dataset contains biological measurements of penguins such as:
  - Species
  - Island
  - Bill length and depth
  - Flipper length
  - Body mass
  - Sex

## Tools & Technologies
- Python
- Jupyter Notebook
- Pandas
- Matplotlib

## Project Structure
- `code.ipynb` – Main Jupyter Notebook containing data analysis and visualizations
- `penguins.csv` – Dataset used for analysis
- `COSM PROJECT REPORT.docx` – Project documentation and observations

## Key Features
- Loaded and explored structured CSV data using Pandas
- Performed data cleaning and handling of missing values
- Analyzed relationships between different attributes
- Visualized data using plots and charts for better interpretation
